# Certification Showcase Website

A responsive certification showcase website built to organize and display academic, internship, and course certificates.

## Live Demos
- Neocities: https://yokeshv.neocities.org/mycertificates/
- GitHub Pages: https://yokeshvenkadachalam.github.io/my-certifications/

## Features
- Responsive layout using Bootstrap
- Clean folder structure with relative asset paths
- Hover effects and smooth scrolling
- Deployment-safe for GitHub Pages and other hosting platforms

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Bootstrap

## Purpose
This project was built to practice frontend structuring, asset management, and real-world deployment workflows.
